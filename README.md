## 🤝 ¿Cómo contribuir?

¡Las contribuciones son bienvenidas! Si encuentras errores, tienes sugerencias o quieres agregar nuevas guías:

1. **Fork** el repositorio
2. Crea una **branch** para tu contribución (`git checkout -b feature/nueva-guia`)
3. Realiza tus cambios y **commit** (`git commit -m 'Agrega guía de X'`)
4. Haz **push** a la branch (`git push origin feature/nueva-guia`)
5. Abre un **Pull Request**

### Tipos de contribuciones:

- 🐛 Corrección de errores
- ✨ Nuevas guías
- 📝 Mejoras en la documentación existente
- 🌍 Traducciones
- 💡 Sugerencias y feedback
